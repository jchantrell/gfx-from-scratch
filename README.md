# gfx-from-scratch

This is a recreational project I am using to learn more about graphical rendering. I chose to use Rust and WebAssembly simply because I was curious and wanted to experiment with them.

Implementation based on the book [Computer Graphics from Scratch](https://gabrielgambetta.com/computer-graphics-from-scratch/index.html). If the topic interests you I highly rate it.

Run `pnpm dev` to start the dev server in watch mode.
